### 课程简介

互+班金融方向的大二下开课，延续金融学基础内容，主要讲授金融衍生工具（包括期权、期货以及如何建立期权期货的数学计算模型）的内容

- 难易程度：总体较难，非常抽象，数学多，概念复杂。邓老师考试大题以原题为主。
